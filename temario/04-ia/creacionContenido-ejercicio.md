# Ejercicio de Creación de Contenido Didáctico con IA

## Objetivo

Generar materiales didácticos para enseñanza del lenguaje digital usando IA.

## Nivel: Medio

## Corpus de trabajo

Glosario completo + campos semánticos de ejercicios anteriores


## Parte 1: Generación de ejercicios por nivel (25 min)

**Objetivo:** Crear ejercicios diferenciados para distintos niveles (A1-B2).

```markdown
Prompt para ejercicios de vocabulario:

"Genera un ejercicio de vocabulario sobre términos del lenguaje digital
para nivel [NIVEL] del MCER.

El ejercicio debe incluir:
1. 10 términos del lenguaje digital apropiados para este nivel
2. Instrucciones claras para el estudiante
3. Solucionario

Tipos de ejercicio (elige uno apropiado para el nivel):
   - Nivel A1-A2: Emparejar términos con definiciones simples
   - Nivel B1: Completar espacios en blanco con contexto
   - Nivel B2: Definir términos y escribir oraciones de ejemplo

Términos a usar (selecciona los apropiados para este nivel):
influencer, postear, tuitear, viral, hashtag, trending topic,
community manager, engagement, linkear, story, feed, like,
share, follower, retuitear, bloquear, menciones, notificación"


Ejecutar para 3 niveles:

NIVEL A2:
Prompt: ___
Resultado: ___
¿Apropiado para principiantes? ___

NIVEL B1:
Prompt: ___
Resultado: ___
¿Equilibrado en dificultad? ___

NIVEL B2:
Prompt: ___
Resultado: ___
¿Suficientemente desafiante? ___
```

## Parte 3: Creación de infografías conceptuales (20 min)

**Objetivo:** Generar contenido para infografías sobre campos semánticos.

```markdown
Prompt para contenido de infografía:

"Genera el contenido para una infografía sobre el campo semántico
de '[CAMPO]' en el lenguaje digital.

Estructura:
1. Título atractivo
2. Definición del campo (2-3 frases)
3. 8-10 términos clave relacionados
4. Para cada término:
   - Término
   - Definición breve (1 frase)
   - Un ejemplo de uso
5. Relaciones entre términos (sinónimos, antónimos, relacionados)

Campos posibles:
- Redes sociales (influencer, follower, engagement...)
- Creación de contenido (postear, linkear, viralizar...)
- Plataformas (tweet, story, reel, feed...)

Usa lenguaje visual y fácil de entender para estudiantes."
```

## Parte 4: Generación de cuestionarios de comprensión (15 min)

**Objetivo:** Crear preguntas de comprensión sobre textos del lenguaje digital.

```markdown
Prompt:

"Genera un cuestionario de comprensión lectora sobre este texto
para estudiantes de español nivel B1.

Texto:
[TEXTO SOBRE LENGUAJE DIGITAL]

El cuestionario debe incluir:
1. 5 preguntas de choice múltiple (4 opciones)
2. 3 preguntas de respuesta corta
3. 2 preguntas de opinión/discusión
4. Solucionario con explicaciones

Las preguntas deben evaluar:
- Comprensión de vocabulario técnico
- Comprensión de ideas principales
- Inferencia de significado por contexto"


Texto a usar:
"Los influencers son personas que han construido una audiencia
en redes sociales gracias a su contenido especializado. Cuando
una marca quiere promocionar un producto, puede colaborar con
un influencer para que lo mencione en sus posts. Si el contenido
resuena con la audiencia, puede volverse viral y alcanzar a
miles de personas. El éxito se mide en engagement: likes,
comentarios y shares."

Resultado del cuestionario:
___

¿Las preguntas son apropiadas para nivel B1? ___
¿El solucionario es correcto? ___
```

## Parte 6: Validación pedagógica (10 min)

**Objetivo:** Establecer criterios para evaluar calidad del contenido generado.

```markdown
Criterios de validación:

Para cada ejercicio generado, evaluar:

1. Adecuación de nivel
   - ¿El vocabulario es apropiado para el nivel indicado?
   - ¿La complejidad gramatical es correcta?

2. Precisión lingüística
   - ¿Las definiciones son correctas?
   - ¿Los ejemplos son auténticos?

3. Claridad de instrucciones
   - ¿Un estudiante entendería qué hacer?
   - ¿Hay ambigüedades?

4. Valor pedagógico
   - ¿El ejercicio enseña algo útil?
   - ¿Es practicable en un aula?

5. Solucionario
   - ¿Las respuestas son correctas?
   - ¿Hay posibles respuestas alternativas válidas?

Ejercicio a evaluar (seleccionar uno de los anteriores):
___

Evaluación según criterios:

2. Precisión lingüística: ___ / 10
3. Claridad de instrucciones: ___ / 10
4. Valor pedagógico: ___ / 10
5. Solucionario: ___ / 10

Total: ___ / 50

Comentarios de mejora:
___
```
